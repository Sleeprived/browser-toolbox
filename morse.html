Morse Code Studio

Translate text to International Morse Code and back — then hear it, save it as audio, see it flash, or feel it buzz. Everything runs in your browser.

Tap it in

Tap the pad below — or turn on keyboard tapping and press the key: short tap = dot, long hold = dash. Pause briefly to end a letter, pause longer to start a new word. Press Esc to stop keyboard tapping. Your taps land in the input above as Morse.
