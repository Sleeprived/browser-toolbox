Browser Toolbox

Fourteen handy tools that run entirely in your browser.

🔒 100% private. Nothing you type or drop here is ever uploaded. No account, no tracking, works offline once loaded.
QR Code Studio
Make QR codes for text, links, WiFi logins, and contact cards. Download PNG or SVG.
EXIF Cleaner
Strip GPS, camera, and timestamp metadata from JPEG & PNG photos. No upload.
Passphrase Generator
Strong diceware passphrases plus an offline password strength meter.
Password Vault
A local, offline password manager. AES-256 encrypted file you keep — with TOTP, tags, and a generator.
CSV ⇆ JSON Workbench
View, sort, clean, and convert CSV and JSON in a table. Nothing leaves the page.
Color Palette Extractor
Pull the dominant colors from any image. Export as hex, CSS variables, or JSON.
Cron Explainer
Plain-English cron, a field-by-field breakdown, a firing heatmap, and the next run times.
Encode / Decode Multitool
Convert text to and from Base64, Hex, URL encoding, HTML entities, and Binary.
JWT Decoder
Decode and inspect JSON Web Tokens; verify HS256/384/512 signatures.
Image Resizer
Resize and compress JPEG, PNG, and WebP images. Rotation auto-fixed.
Morse Code Studio
Translate text to Morse and back, then play it, save it as audio, flash it, or vibrate it.
Cipher Studio
Encode & decode classic secret codes: Tap Code, Baconian, Pigpen, and Semaphore.
TOTP / 2FA Generator
Generate time-based 2FA codes from a secret or otpauth:// URI. Nothing is stored.
Hash & Checksum Verifier
SHA-1/256/384/512 of text or a file; verify a download against its checksum.
